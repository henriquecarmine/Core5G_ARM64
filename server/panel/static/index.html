Condições do Canal
5G NR n78 · TDD · 3.3–3.8 GHz · BW 100 MHz · SCS 30 kHz · 66 PRBs  ·  ↓ DL ~665 Mbps  ·  ↑ UL ~250 Mbps
PL(d) = 13.54 + 39.08·log₁₀(d) + 20·log₁₀(f_c) − 0.6·(h_UT−1.5) [3GPP TR 38.901 · UMa NLOS · f_c = 3.5 GHz · h_UT = 1.5 m]
PL(d) = Path Loss em dB (atenuação total do sinal) d = distância UE ↔ gNB em metros f_c = frequência portadora em GHz (3.5 GHz = banda n78) h_UT = altura do UE acima do solo em metros (padrão: 1.5 m) UMa = Urban Macro (macro-célula urbana, h_BS = 25 m) NLOS = Non-Line-of-Sight (sem visada direta UE↔antena)
SINR = C_signal / (N₀ + I) → C_max = B · log₂(1 + SINR) [Shannon–Hartley · B = 100 MHz · 5G NR n78 TDD]
SINR = Signal-to-Interference-plus-Noise Ratio (dB) C_signal = potência do sinal útil recebido (dBm) N₀ = piso de ruído térmico (kTB, ≈ −100 dBm / 100 MHz) I = potência total da interferência co-canal (dBm) B = largura de banda do canal = 100 MHz (n78 TDD) C_max = capacidade máxima teórica do canal (Mbps) C/I = Carrier-to-Interference ratio (nível de qualidade do sinal)
